## **Assignment Overview**

In this module, we covered peer code review. In this assignment, you are required to create a pull request and work on a code review.  Then you will be asked to peer-review a piece of code and will receive feedback and revised code for further review.

---

## **Procedure**

Complete Quest 5 in your OSS Doorway repository. You will be required to:

1. Create a piece of code according to the requirements

2. Work on improving your contribution according to the code reviews received

3. Then, you will be the one reviewing a piece of code

---

### **What to submit?**

After you finish **Quest 5**, submit the URL of your repository.
