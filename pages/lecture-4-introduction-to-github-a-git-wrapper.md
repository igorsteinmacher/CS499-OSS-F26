# Lecture 4. Introduction to GitHub (a git wrapper)


## **Summary**

In this lecture, we will explore the differences between git and GitHub and the basic features of GitHub so we can move ahead and work with these tools in our course.

**Topics**

- Connecting git to our GitHub repository
- add, commit, status, log... the basics of git
- branching and merging
- moving the commits and HEAD
